# Philippine Air Carriers Flight Records Analysis

This repository provides a comprehensive analytical pipeline for analyzing
Philippine air carrier flight records using data science, statistics, and
machine learning techniques.

Dataset Source:
https://www.kaggle.com/datasets/bwandowando/philippine-air-carriers-flight-records

## Objectives
- Understand flight activity patterns
- Quantify airline performance and reliability
- Identify delay and cancellation drivers
- Analyze route networks and airport connectivity
- Develop predictive models for operational forecasting
- Build decision-support dashboards

## Tools & Libraries
- Python 3.10+
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- NetworkX
- scikit-learn, statsmodels
- Prophet (optional)
